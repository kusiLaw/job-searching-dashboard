# job-searching-dashboard

A free, open source, single-file web app for tracking every application from first save to final offer.

## What it does:
1. Drag-and-drop kanban board (Saved → Applied → Interview → Offer)
2. Searchable, filterable, sortable list view
3. Smart follow-up flag — highlights roles you applied to 14+ days ago with no reply
4. Salary comparison chart across active applications
5. Stores job post URLs so you never lose the original listing

No sign-up. No data leaves your browser. Just download and open.
